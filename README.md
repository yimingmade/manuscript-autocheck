## Install

Install from this repository after it is pushed to GitHub:

```bash
python3 install-skill-from-github.py --repo OWNER/REPO --path skills/manuscript-presubmission-checklist-revision
```

Or copy the skill directory directly:

```bash
cp -R skills/manuscript-presubmission-checklist-revision ~/.codex/skills/
```

Restart Codex after installation.
